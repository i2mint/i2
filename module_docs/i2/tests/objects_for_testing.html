This function is meant to demo several combinations of type declaration (in signature or/and in docs), as well as with or without defaults.

Parameters:

  • any_var – no type declared in either signature of doc

  • a_list (list) – type declared in signature AND doc

  • a_dict – type declared in signature but NOT in doc

  • an_int (int) – type declared in doc but NOT in signature

  • a_float (float) – type declared in signature AND doc, and has a default of the desired type

  • a_bool – type declared in signature but NOT in doc, and has a default of the desired type

  • a_str (str) – type declared in signature AND doc, but has a default that is not of that type

line description :return str: Will just return the ‘test_func_1 returned’ string :tags foo, bar
