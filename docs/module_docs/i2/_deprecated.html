class i2._deprecated.Command(func, *args, **kwargs)[source]¶

A class that holds a (caller, args, kwargs) triple and allows one to execute caller(*args, **kwargs)

Parameters

  • func – A callable that will be called with (*args, **kwargs) argument.

  • args – The positional arguments to call the func with.

  • kwargs – The keyword arguments to call the func with.

>>> c = Command(print, "hello", "world", sep=", ")
>>> c()
hello, world

What happens (when a command is executed) if some of the arguments are commands themselves? Well, the sensible thing happens. These commands are executed. You can use this to define, declaratively, some pretty complex instructions, and only fetch the data you need and execute everything, once you’re ready.

>>> def show(a, b):
...     print(f"Showing this: {a=}, {b=}")
>>> def take_five():
...     return 5
>>> def double_val(val):
...     return val * 2
>>> command = Command(
...     show,
...     Command(take_five),
...     b=Command(double_val, 'hello'),
... )
>>> command
Command(show, Command(take_five), b=Command(double_val, 'hello'))
>>> command()
Showing this: a=5, b='hellohello'

Of course, as your use of Command gets more complex, you may want to subclass it and include some “validation” and “compilation” in the init.

The usual way to call a function is to… erm… call it. But sometimes you want to do things differently. Like validate it, put it on a queue, etc. That’s where specifying a different _caller will be useful.

>>> class MyCommand(Command):
...     def _caller(self):
...         f, a, k = self.func, self.args, self.kwargs
...         print(f"Calling {f}(*{a}, **{k}) with result: {f(*a, **k)}")
...
>>> c = MyCommand(print, "hello", "world", sep=", ")
>>> c()
hello, world
Calling <built-in function print>(*('hello', 'world'), **{'sep': ', '}) with result: None
classmethod curried(func, **kw_defaults)[source]¶

Get an Command maker for a specific function, with defaults and signature!

>>> def foo(x: str, y: int):
...     return x * y
...
>>> foo('hi', 3)
'hihihi'
>>>
>>> foo_command = Command.curried(foo, y=2)
>>> Sig(foo_command)
<Sig (x: str, y: int = 2)>
>>> f = foo_command('hi', y=4)
>>> f()
'hihihihi'
>>> ff = foo_command('hi')
>>> ff
Command(foo, 'hi')
>>> ff()
'hihi'
