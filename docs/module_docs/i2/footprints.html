i2.footprints.attrs_used_by_method(method, remove_duplicates=True)[source]¶

Extracts a list of cls attributes which are used by a method or method_name function :param method: The method (object) to analyze

Returns

A list of attribute names (of the class or instance thereof) that are accessed in the code of the said method.

Example:

Consider the method A.target_method coming from the following code in i2.tests.footprints: ```python def func(obj):

“””Accesses attributes ‘a’ and ‘b’ of obj””” return obj.a + obj.b

class A:

e = 2

def __init__(self, a=1, b=0, c=1, d=10):

self.a = a self.b = b self.c = c self.d = d

def target_method(self, x):

“””Accesses [‘a’, ‘b’, ‘c’, ‘e’]””” t = func(self) # and this function will access some attributes! tt = self.other_method(t) return x * tt / self.e

def other_method(self, x=1):

“””Accesses [‘c’, ‘e’]””” w = self.c * 2 # c is accessed first return self.e + self.c * x - w # and c is accessed again

@classmethod def a_class_method(cls, y):

“””Accesses [‘e’]””” return cls.e + y

```

>>> from i2.tests.footprints import A
>>> assert attrs_used_by_method(A.target_method) == {'a', 'b', 'c', 'e'}
